# Auth

теперь мне нужен механизм авторизации. он не мультиюзерский, и зависит от конфига. в конфиге перечислены сети, доступ из которых идет без авторизации. для остальных сетей показываем окно авторизации для ввода пароля (если он вообще задан в настройках).

при правильном пароле открывается весь функционал. при неправильном ничего.

пароль и списки сетей хранятся в базе. и пока не пишем форму настройки доступа
